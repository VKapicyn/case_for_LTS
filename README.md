# case_for_LTS

Создать класс security.
Заложить в класс все необходимые поля, свойственные ценной бумаге (последняя цена, название и тд.)
Предусмотреть в возможность хранения исторических цен в экземпляре класса security. 
Сделать на стандартной визуальной форме все необходимые поля ввода данных для заявки.
Заложить в класс метод-заглушку, принимающий все указанные на форме свойства и возвращающий строку, содержащую указанные значения для дальнейшего вывода messageBox с запросом подтверждения транзакции.

Найти доступный источник исторических или текущих данных  по ценной бумаге (к примеру библитека Atentis http://www.alorbroker.ru/tools/atentis/ )  
Сделать модуль загрузки данных, создать на основе данных несколько объектов security, посчитать и вывести в файл историческую корреляцию по выбранным инструментам  (от 1 дня до месяца, в зависимости от выбранного временного шага).

Парный трейдинг. Найти пары на российском и зарубежном рынках. Автоматизировать поиск пар на зарубежке.
